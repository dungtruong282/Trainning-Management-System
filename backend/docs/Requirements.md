# Requirements

## Project Name

Training Management System

## Project Description

Training Management System is an internal web application that helps enterprises manage employee training, courses, lessons, quizzes, and learning progress.

## Actors

- Admin
- HR
- Trainer
- Employee

## Functional Requirements

### Admin

- Manage departments
- Manage users
- Manage courses
- View reports

### HR

- Manage employees
- Assign courses

### Trainer

- Create courses
- Create lessons
- Upload documents
- Create quizzes

### Employee

- Login
- View assigned courses
- Study lessons
- Complete quizzes
- View learning progress

## Technology

- ASP.NET Core Web API
- SQL Server
- Entity Framework Core
- React (Future)
